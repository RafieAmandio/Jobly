# Jobly

Read `docs/PRD.md` before making any changes. It contains the full product requirements, technical architecture, database schema, API integrations, and development roadmap.

## Stack

- Python 3.12+ with uv
- aiogram v3 (Telegram bot)
- SQLAlchemy 2.0 async + asyncpg (Supabase PostgreSQL)
- arq + Redis (task queue)
- APScheduler v4 (scheduled jobs)
- Kimi 2.6 via OpenAI-compatible SDK (AI)
- Xendit REST API (payments)
- python-docx + WeasyPrint (document generation)

## Target Market

Indonesia — bilingual (Bahasa Indonesia + English)
